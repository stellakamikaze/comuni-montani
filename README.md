# Casa in quota — Dashboard comuni montani >1200m

Mappa interattiva per esplorare comuni sopra i 1200 m in Piemonte, Valle d'Aosta e Lombardia
come possibile luogo dove vivere / seconda casa. 30 comuni con prezzi €/mq, servizi, accessibilità,
esposizione solare e punteggi.

- **Live (protetta da password)**: vedi GitHub Pages di questo repo, cartella `/site`.
- Sorgente: `template.html` + `dati.json` → `build.js` genera `index.html`, poi cifrato con staticrypt.
